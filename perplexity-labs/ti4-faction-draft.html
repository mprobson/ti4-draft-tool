TI4 Faction Draft Helper

Secretly deal each player 1 random faction from PoK + Keleres, then 3 more random factions from the remaining pool. The hidden draft is kept on each player’s device with a local secret code.

Setup

Everyone entering the same code will reproduce the same randomized draft. Use a fresh code each session.

How to use

  1. Open this HTML file directly in a browser.
  2. Enter the number of players, names, and a shared seed.
  3. Click Generate draft.
  4. Copy your secret link and send it to each player if you want only their hand shown.

Secret links use a URL hash, so they stay offline and never leave the browser.
